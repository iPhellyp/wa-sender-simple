# Boas praticas anti-spam

- Use base propria ou autorizada.
- Evite conteudo enganoso.
- Permita descadastro.
- Respeite respostas dos contatos.
- Aqueca numero novo com baixo volume.
- Nao envie alto volume em numero novo.
- Comece com 2 a 5 contatos.
- Depois avance para 10 a 20 contatos.
- Aumente gradualmente.
- Evite mensagens repetidas.
- Evite listas frias.
- Use intervalos entre envios.
- Monitore falhas, bloqueios e respostas.
- Nao prometa disparo ilimitado.
- Use spintax com responsabilidade para evitar mensagens identicas.
- Sempre envie campanha teste pequena antes de aumentar volume.
